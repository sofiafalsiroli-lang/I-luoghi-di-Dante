# Wireframe: City/Region Secondary Page Layout


```
+---------------------------------------------------------------+
| Sulle tracce di Dante (main head, always at top)              |
+---------------------------------------------------------------+
| [Menu bar: HOME | I LUOGHI DI DANTE | MAPPA | ABOUT | LANG]   |
+---------------------------------------------------------------+
| City/region name (page head, e.g. "Bologna")                 |
+---------------------------------------------------------------+
|                                                               |
| +--------------------------+   +------------------------+     |
| |                          |   |                        |     |
| |  City/region text        |   |   Interactive map      |     |
| |  (narrative,             |   |   (Leaflet, points     |     |
| |  quotations,             |   |   highlight as text    |     |
| |  images inline)          |   |   scrolls or is        |     |
| |                          |   |   clicked)             |     |
| +--------------------------+   +------------------------+     |
|   (wider text column,          (narrower map column)          |
|    e.g. 2/3 : 1/3)                                            |
|                                                               |
+---------------------------------------------------------------+
| Footer (blank for now)                                        |
+---------------------------------------------------------------+
```


## Details
- **Page head:**
  - City/region name (e.g. "Bologna") appears under the menu bar and above the content columns.
- **Left column (wider, ~2/3):**
  - Narrative text, quotations, and images (images appear at relevant points, not too large, with captions)
  - For EN/DE: "TRADUZIONE MANCANTE" in red where translation is missing
- **Right column (narrower, ~1/3):**
  - Interactive map (with points that highlight as you scroll/click related text/images)
- **Responsive:**
  - On mobile, map stacks above text
