# Tutorial: about_de.html

## Introduction
`about_de.html` is the **German-language "About the Website" page** (*Über die Website*) of the *Dante's Journey* project. It explains the academic context in which the website was created, describes how the code, texts, and quotes were produced (including the role of AI), and includes an important disclaimer about the educational — rather than scholarly — purpose of the site. Like `index_de.html`, it uses the shared external stylesheet (`css/style.css`) for site-wide styles and a small **internal `<style>` block** for rules specific to this page's text area. The page uses several **semantic HTML elements** (`<article>`, `<section>`, `<main>`) and a `<script>` tag at the bottom of `<body>` for the language-switching feature.
