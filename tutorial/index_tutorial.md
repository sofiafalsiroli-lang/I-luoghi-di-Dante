# Tutorial: index.html

## Introduction
This tutorial explains the structure and function of the `index.html` file, which is the **entry point** of the *Dante's Journey* website. Its sole purpose is to detect the visitor and let them choose their preferred language — Italian, English, or German — before redirecting them to the corresponding version of the site. Because it serves as a language-selection gateway rather than a full content page, it is intentionally minimal: it contains a small amount of HTML and all of its styling is written directly inside the file using a `<style>` block (called *internal CSS*), with no external stylesheet.

| Code | Explanation (English) | Spiegazione (Italiano) |
|------|-----------------------|------------------------|
| `<!DOCTYPE html>` | This is the very first line of every HTML file. It is not a tag but a **declaration** that tells the browser: "This document is written in HTML5." Without it, some browsers may display the page incorrectly. | È la primissima riga di ogni file HTML. Non è un tag ma una **dichiarazione** che dice al browser: "Questo documento è scritto in HTML5." Senza di essa, alcuni browser potrebbero visualizzare la pagina in modo errato. |
| `<html lang="en"> … </html>` | The `<html>` element is the **root** (outermost container) of the entire page. Everything else lives inside it. The attribute `lang="en"` tells the browser and screen readers that the default language of this particular page is English. | L'elemento `<html>` è la **radice** (il contenitore più esterno) dell'intera pagina. Tutto il resto si trova al suo interno. L'attributo `lang="en"` comunica al browser e agli screen reader che la lingua predefinita di questa pagina è l'inglese. |
| `<head> … </head>` | The `<head>` section is **not visible** on screen. It contains metadata (information *about* the page) and resources the browser needs before displaying anything, such as the page title, character encoding, viewport settings, the favicon, and CSS styles. | La sezione `<head>` **non è visibile** sullo schermo. Contiene i metadati (informazioni *sulla* pagina) e le risorse di cui il browser ha bisogno prima di visualizzare qualsiasi cosa: il titolo, la codifica dei caratteri, le impostazioni del viewport, il favicon e gli stili CSS. |
| `<meta charset="UTF-8">` | A `<meta>` tag provides metadata. The `charset="UTF-8"` attribute sets the **character encoding** to UTF-8, which supports virtually all characters from all languages (including accented letters like à, è, ü). Without this, special characters may appear as garbled symbols. | Il tag `<meta>` fornisce metadati. L'attributo `charset="UTF-8"` imposta la **codifica dei caratteri** su UTF-8, che supporta quasi tutti i caratteri di tutte le lingue (comprese le lettere accentate come à, è, ü). Senza di esso, i caratteri speciali potrebbero apparire come simboli incomprensibili. |
| `<title>Dante's Journey - Language Selection</title>` | The `<title>` tag sets the **text shown in the browser tab** (or window title bar). Search engines also use this text as the headline in search results. | Il tag `<title>` imposta il **testo visualizzato nella scheda del browser** (o nella barra del titolo della finestra). I motori di ricerca usano questo testo come titolo nei risultati di ricerca. |
| `<meta name="viewport" content="width=device-width, initial-scale=1.0">` | This `<meta>` tag controls how the page is displayed on mobile devices. `width=device-width` makes the page width match the device's screen width, and `initial-scale=1.0` sets the initial zoom level to 100%. Without this, the page would appear very small on smartphones. | Questo tag `<meta>` controlla come la pagina viene visualizzata sui dispositivi mobili. `width=device-width` fa sì che la larghezza della pagina corrisponda alla larghezza dello schermo del dispositivo, mentre `initial-scale=1.0` imposta lo zoom iniziale al 100%. Senza di esso, la pagina apparirebbe molto piccola sugli smartphone. |
| `<link rel="icon" type="image/svg+xml" href="data/pictures/favicon.svg">` | The `<link>` tag here loads the **favicon** — the small icon that appears in the browser tab next to the page title. The `rel="icon"` attribute identifies its purpose, `type="image/svg+xml"` specifies the file format (SVG), and `href="data/pictures/favicon.svg"` is the **path** (address) to the icon file. | Il tag `<link>` carica il **favicon** — la piccola icona che appare nella scheda del browser accanto al titolo della pagina. L'attributo `rel="icon"` ne identifica lo scopo, `type="image/svg+xml"` specifica il formato del file (SVG) e `href="data/pictures/favicon.svg"` è il **percorso** (indirizzo) del file dell'icona. |
| `<style> … </style>` | The `<style>` block contains **internal CSS** — style rules written directly inside the HTML file rather than in a separate `.css` file. CSS (Cascading Style Sheets) controls the visual appearance of the page: colours, fonts, sizes, spacing, and layout. | Il blocco `<style>` contiene il **CSS interno** — regole di stile scritte direttamente nel file HTML anziché in un file `.css` separato. Il CSS (Cascading Style Sheets) controlla l'aspetto visivo della pagina: colori, caratteri, dimensioni, spaziature e layout. |
| `body { font-family: sans-serif; background: #f8f8f8; color: #222; display: flex; flex-direction: column; align-items: center; justify-content: center; height: 100vh; margin: 0; }` | This CSS rule styles the `<body>` element (the whole visible area). `font-family: sans-serif` sets a clean, modern font. `background: #f8f8f8` gives a very light grey background. `color: #222` sets the default text colour to near-black. The `display: flex; flex-direction: column; align-items: center; justify-content: center` properties use **Flexbox** to centre the content both horizontally and vertically. `height: 100vh` makes the body fill the full viewport height (100% of the screen). `margin: 0` removes the default spacing around the page edges. | Questa regola CSS stilizza l'elemento `<body>` (tutta l'area visibile). `font-family: sans-serif` imposta un carattere pulito e moderno. `background: #f8f8f8` dà uno sfondo grigio molto chiaro. `color: #222` imposta il colore del testo predefinito su quasi-nero. Le proprietà `display: flex; flex-direction: column; align-items: center; justify-content: center` usano il **Flexbox** per centrare il contenuto sia orizzontalmente che verticalmente. `height: 100vh` fa sì che il body occupi l'intera altezza del viewport (100% dello schermo). `margin: 0` rimuove la spaziatura predefinita attorno ai bordi della pagina. |
| `.lang-select { background: #fff; padding: 2em 3em; border-radius: 12px; box-shadow: 0 2px 12px rgba(0,0,0,0.08); text-align: center; }` | This rule styles the white card that contains the language buttons. `.lang-select` is a **class selector** — it applies to any element that has `class="lang-select"`. `background: #fff` gives it a white background. `padding: 2em 3em` adds inner spacing (2em top/bottom, 3em left/right). `border-radius: 12px` rounds the corners. `box-shadow` adds a subtle drop shadow to make the card "float". `text-align: center` centres all text inside. | Questa regola stilizza il riquadro bianco che contiene i pulsanti delle lingue. `.lang-select` è un **selettore di classe** — si applica a qualsiasi elemento che ha `class="lang-select"`. `background: #fff` gli dà uno sfondo bianco. `padding: 2em 3em` aggiunge una spaziatura interna (2em sopra/sotto, 3em sinistra/destra). `border-radius: 12px` arrotonda gli angoli. `box-shadow` aggiunge un'ombra leggera per fare "galleggiare" il riquadro. `text-align: center` centra tutto il testo all'interno. |
| `.lang-select h1 { margin-bottom: 1em; font-size: 1.5em; }` | Styles the `<h1>` heading inside the card. `margin-bottom: 1em` adds space below the heading. `font-size: 1.5em` makes it 1.5 times larger than the base font size. | Stilizza il titolo `<h1>` all'interno del riquadro. `margin-bottom: 1em` aggiunge spazio sotto il titolo. `font-size: 1.5em` lo rende 1,5 volte più grande rispetto alla dimensione base del carattere. |
| `.lang-links { display: flex; gap: 1.5em; justify-content: center; margin-top: 1em; }` | Styles the container that holds the three language links. `display: flex` lays out the links in a horizontal row. `gap: 1.5em` adds space between each link. `justify-content: center` centres them horizontally. `margin-top: 1em` adds space above the row of links. | Stilizza il contenitore che racchiude i tre link delle lingue. `display: flex` dispone i link in una riga orizzontale. `gap: 1.5em` aggiunge spazio tra ciascun link. `justify-content: center` li centra orizzontalmente. `margin-top: 1em` aggiunge spazio sopra la riga dei link. |
| `.lang-links a { text-decoration: none; color: #b22222; font-weight: bold; font-size: 1.1em; border: 1px solid #b22222; border-radius: 6px; padding: 0.5em 1.2em; transition: background 0.2s, color 0.2s; }` | Styles each language link (`<a>` tag) inside `.lang-links`. `text-decoration: none` removes the default underline. `color: #b22222` sets the text to a deep red (matching the Dante theme). `font-weight: bold` makes the text bold. `font-size: 1.1em` slightly enlarges the text. `border: 1px solid #b22222` draws a red border around each button. `border-radius: 6px` rounds the corners. `padding: 0.5em 1.2em` adds inner spacing so the button is not too tight. `transition: background 0.2s, color 0.2s` creates a smooth 0.2-second animation when the colours change on hover. | Stilizza ogni link di lingua (tag `<a>`) all'interno di `.lang-links`. `text-decoration: none` rimuove la sottolineatura predefinita. `color: #b22222` imposta il testo su un rosso scuro (coerente con il tema dantesco). `font-weight: bold` rende il testo in grassetto. `font-size: 1.1em` ingrandisce leggermente il testo. `border: 1px solid #b22222` disegna un bordo rosso attorno a ogni pulsante. `border-radius: 6px` arrotonda gli angoli. `padding: 0.5em 1.2em` aggiunge spaziatura interna. `transition: background 0.2s, color 0.2s` crea un'animazione fluida di 0,2 secondi quando i colori cambiano al passaggio del mouse. |
| `.lang-links a:hover { background: #b22222; color: #fff; }` | The `:hover` **pseudo-class** applies a style only when the user moves the mouse cursor over the element. When hovering over a language button, the background fills with red (`#b22222`) and the text turns white (`#fff`), inverting the colours for a clear visual feedback. | La **pseudo-classe** `:hover` applica uno stile solo quando l'utente passa il cursore del mouse sull'elemento. Quando si passa sopra un pulsante di lingua, lo sfondo diventa rosso (`#b22222`) e il testo diventa bianco (`#fff`), invertendo i colori per un chiaro feedback visivo. |
| `<body> … </body>` | The `<body>` element contains all the **visible content** of the page — everything the user actually sees in the browser window. | L'elemento `<body>` contiene tutti i **contenuti visibili** della pagina — tutto ciò che l'utente vede nella finestra del browser. |
| `<div class="lang-select"> … </div>` | A `<div>` is a generic **container** element used to group other elements together. This `<div>` has `class="lang-select"`, so the CSS rules for `.lang-select` apply to it, giving it the white card appearance. | Un `<div>` è un elemento **contenitore** generico usato per raggruppare altri elementi. Questo `<div>` ha `class="lang-select"`, quindi le regole CSS per `.lang-select` si applicano ad esso, conferendogli l'aspetto del riquadro bianco. |
| `<h1>Select your language / Seleziona la lingua / Sprache wählen</h1>` | The `<h1>` tag defines a **level-1 heading**, the most important heading on the page. Here it shows the invitation to select a language written in all three supported languages (English, Italian, German), so every visitor understands it regardless of their language. | Il tag `<h1>` definisce un **titolo di primo livello**, il più importante della pagina. Qui mostra l'invito a selezionare una lingua scritto nelle tre lingue supportate (inglese, italiano, tedesco), in modo che ogni visitatore lo comprenda indipendentemente dalla propria lingua. |
| `<div class="lang-links"> … </div>` | Another `<div>` container, this time with `class="lang-links"`. It groups the three language links together and uses Flexbox (as defined in the CSS) to display them side by side. | Un altro contenitore `<div>`, questa volta con `class="lang-links"`. Raggruppa i tre link delle lingue e usa il Flexbox (come definito nel CSS) per visualizzarli affiancati. |
| `<a href="it/index_it.html">Italiano</a>` | The `<a>` tag creates a **hyperlink**. The `href` attribute contains the **URL** (the address) the link points to — in this case the Italian version of the homepage, located in the `it/` subfolder. The text between the tags ("Italiano") is what the user sees and clicks. | Il tag `<a>` crea un **collegamento ipertestuale**. L'attributo `href` contiene l'**URL** (l'indirizzo) a cui punta il link — in questo caso la versione italiana della home page, nella sottocartella `it/`. Il testo tra i tag ("Italiano") è ciò che l'utente vede e su cui clicca. |
| `<a href="en/index_en.html">English</a>` | Works exactly like the Italian link above, but points to the English version of the homepage in the `en/` subfolder. | Funziona esattamente come il link italiano sopra, ma punta alla versione inglese della home page nella sottocartella `en/`. |
| `<a href="de/index_de.html">Deutsch</a>` | Works exactly like the two links above, but points to the German version of the homepage in the `de/` subfolder. | Funziona esattamente come i due link sopra, ma punta alla versione tedesca della home page nella sottocartella `de/`. |
