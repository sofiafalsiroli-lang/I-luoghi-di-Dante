# Tutorial: bologna_de.html

## Introduction
`bologna_de.html` is the **German-language city page for Bologna**. It follows the same structure as all other city pages on the site: a language bar, a site banner, a navigation bar, and then the main content divided into thematic sections — each combining text, a quote from the *Divina Commedia*, and one or more images. This page is the most complex in the site because it has a **rich internal `<style>` block** with many custom CSS classes for layout (flexbox columns, responsive behaviour, image tooltip overlays, and decorative left-bordered sections). It also includes an **inline JavaScript block** at the bottom of `<body>` for the image-credit tooltip interaction. Understanding this file gives a thorough overview of how a full content page is built.
