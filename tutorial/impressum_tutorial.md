# Tutorial: impressum.html

## Introduction
`impressum.html` is the **legal notice page** (also called *Impressum* in German-speaking legal tradition) of the *Dante's Journey* website. It is required by Austrian and German law for publicly accessible websites and provides visitors with information about who is responsible for the content, how to contact the author, and a disclaimer about the use of images and texts. Unlike the minimal `404.html`, this page uses **both** an external stylesheet (`css/style.css`, which provides site-wide styles) **and** a small internal `<style>` block for layout rules specific to this page alone. It also loads a JavaScript file at the end of the body.

| Code | Explanation (English) | Spiegazione (Italiano) |
|------|-----------------------|------------------------|
| `<!DOCTYPE html>` | A **declaration** on the very first line that tells the browser to render the page using the modern HTML5 standard. It is not a tag and produces nothing visible. | Una **dichiarazione** sulla primissima riga che indica al browser di visualizzare la pagina usando lo standard moderno HTML5. Non è un tag e non produce nulla di visibile. |
| `<html lang="en"> … </html>` | The **root element** wrapping the entire document. The attribute `lang="en"` declares English as the page's language, helping screen readers and search engines. | L'**elemento radice** che racchiude l'intero documento. L'attributo `lang="en"` dichiara l'inglese come lingua della pagina, aiutando screen reader e motori di ricerca. |
| `<head> … </head>` | The **invisible** section that contains metadata and resources needed before the browser renders the visible content: character encoding, title, viewport setting, favicon, and stylesheets. | La sezione **invisibile** che contiene i metadati e le risorse necessarie prima che il browser visualizzi il contenuto: codifica dei caratteri, titolo, impostazioni del viewport, favicon e fogli di stile. |
| `<meta charset="UTF-8">` | Sets the **character encoding** to UTF-8 so that all characters — including German umlauts (ä, ö, ü) and Italian accents — display correctly. | Imposta la **codifica dei caratteri** su UTF-8, così tutti i caratteri — compresi le vocali tedesche con l'umlaut (ä, ö, ü) e gli accenti italiani — vengono visualizzati correttamente. |
| `<title>Impressum</title>` | Sets the **text shown in the browser tab**. "Impressum" is the internationally recognised term for a legal notice page, understood by German, Austrian, and Italian audiences alike. | Imposta il **testo nella scheda del browser**. "Impressum" è il termine internazionalmente riconosciuto per una pagina di avviso legale, compreso dal pubblico tedesco, austriaco e italiano. |
| `<meta name="viewport" content="width=device-width, initial-scale=1.0">` | Makes the page **responsive** on mobile devices. `width=device-width` matches the page width to the screen, and `initial-scale=1.0` prevents unwanted zoom on load. | Rende la pagina **responsiva** su dispositivi mobili. `width=device-width` adatta la larghezza della pagina allo schermo e `initial-scale=1.0` impedisce zoom indesiderati al caricamento. |
| `<link rel="icon" type="image/svg+xml" href="data/pictures/favicon.svg">` | Loads the **favicon** (the small icon in the browser tab). `rel="icon"` identifies its role, `type="image/svg+xml"` specifies SVG format, and `href` gives the file path. | Carica il **favicon** (la piccola icona nella scheda del browser). `rel="icon"` ne identifica il ruolo, `type="image/svg+xml"` specifica il formato SVG e `href` fornisce il percorso del file. |
| `<link rel="stylesheet" href="../../css/style.css">` | Links an **external CSS file** (`style.css`) located two levels up in the folder structure (`../../`). This file contains the shared styles used across the whole website — fonts, colours, the navigation bar, etc. — so the Impressum page looks consistent with the rest of the site. | Collega un **file CSS esterno** (`style.css`) situato due livelli sopra nella struttura delle cartelle (`../../`). Questo file contiene gli stili condivisi usati in tutto il sito — caratteri, colori, la barra di navigazione, ecc. — così la pagina Impressum appare coerente con il resto del sito. |
| `<style> … </style>` | An **internal CSS block** containing style rules that apply only to this page. These rules fine-tune the layout of the Impressum container box — something not needed anywhere else on the site, so it does not belong in the external `style.css`. | Un **blocco CSS interno** contenente regole di stile che si applicano solo a questa pagina. Queste regole perfezionano il layout del riquadro Impressum — qualcosa non necessario altrove nel sito, quindi non appartiene al file esterno `style.css`. |
| `.impressum-container { max-width: 700px; margin: 3em auto; background: #fff; border-radius: 12px; box-shadow: 0 2px 8px #bbb; padding: 2.5em 2em; font-size: 1.1em; }` | Styles the white card that contains all the legal text. `max-width: 700px` limits how wide the box can grow so it is comfortable to read on large screens. `margin: 3em auto` adds vertical space above/below the box and centres it horizontally. `background: #fff` gives it a white fill. `border-radius: 12px` rounds the corners. `box-shadow: 0 2px 8px #bbb` adds a light grey drop shadow, making the card appear to float above the page background. `padding: 2.5em 2em` adds inner spacing. `font-size: 1.1em` makes the text slightly larger than the default. | Stilizza il riquadro bianco che contiene tutto il testo legale. `max-width: 700px` limita la larghezza massima del riquadro per una lettura confortevole su schermi grandi. `margin: 3em auto` aggiunge spazio verticale sopra/sotto e centra il riquadro orizzontalmente. `background: #fff` gli dà uno sfondo bianco. `border-radius: 12px` arrotonda gli angoli. `box-shadow: 0 2px 8px #bbb` aggiunge un'ombra grigio chiaro per far "galleggiare" il riquadro. `padding: 2.5em 2em` aggiunge spaziatura interna. `font-size: 1.1em` rende il testo leggermente più grande del predefinito. |
| `h1, h2 { color: #b22222; }` | A rule that targets **both** `<h1>` and `<h2>` headings at once (separated by a comma). It sets their text colour to the deep red (`#b22222`) used as the primary brand colour throughout the site. | Una regola che si applica contemporaneamente ai titoli `<h1>` e `<h2>` (separati da una virgola). Imposta il colore del testo sul rosso scuro (`#b22222`) usato come colore principale del marchio in tutto il sito. |
| `.impressum-label { font-weight: bold; }` | A utility class. Any element with `class="impressum-label"` will be displayed in **bold**. It is defined here for potential use on label text (though no element currently uses it in the visible HTML), following common practice of defining reusable helper classes. | Una classe di utilità. Qualsiasi elemento con `class="impressum-label"` verrà visualizzato in **grassetto**. È definita qui per un possibile uso su testi etichetta, seguendo la pratica comune di definire classi di supporto riutilizzabili. |
| `@media (max-width: 800px) { .impressum-container { padding: 1.2em 0.5em; } }` | A **media query** — a CSS rule that applies only under certain screen conditions. `max-width: 800px` means "when the screen is 800 px wide or narrower" (typically tablets and phones). On those screens the inner padding of the container is reduced from `2.5em 2em` to `1.2em 0.5em` so the box does not feel cramped and uses the available screen space better. | Una **media query** — una regola CSS che si applica solo in determinate condizioni dello schermo. `max-width: 800px` significa "quando lo schermo è largo 800 px o meno" (tipicamente tablet e telefoni). Su quegli schermi il padding interno del contenitore viene ridotto da `2.5em 2em` a `1.2em 0.5em` per non rendere il riquadro troppo stretto. |
| `<body> … </body>` | Contains all the **visible content** of the page — everything the user sees in the browser window. | Contiene tutti i **contenuti visibili** della pagina — tutto ciò che l'utente vede nella finestra del browser. |
| `<div class="impressum-container"> … </div>` | A `<div>` is a generic **container** element. With `class="impressum-container"` the CSS rules defined above are applied to it, giving the white card appearance. All legal content is nested inside this container. | Un `<div>` è un elemento **contenitore** generico. Con `class="impressum-container"` vengono applicate le regole CSS definite sopra, conferendo l'aspetto del riquadro bianco. Tutto il contenuto legale è annidato all'interno di questo contenitore. |
| `<h1>Impressum / Legal Notice</h1>` | The **main heading** of the page (`<h1>` is the highest-level heading). The title is written in two languages ("Impressum" in German/Italian legal tradition, "Legal Notice" in English) so all visitors immediately understand the page's purpose. | Il **titolo principale** della pagina (`<h1>` è il titolo di livello più alto). Il titolo è scritto in due lingue ("Impressum" nella tradizione legale tedesca/italiana, "Legal Notice" in inglese) così tutti i visitatori capiscono immediatamente lo scopo della pagina. |
| `<p>This website is a non-commercial, educational project …</p>` | A `<p>` (paragraph) element with a brief statement clarifying that the site is non-commercial and educational. This is important for legal purposes — it establishes the nature of the project upfront. | Un elemento `<p>` (paragrafo) con una breve dichiarazione che chiarisce che il sito è non commerciale ed educativo. Questo è importante a fini legali — stabilisce la natura del progetto fin dall'inizio. |
| `<h2>Contact</h2>` | An `<h2>` tag creates a **level-2 heading** — a sub-heading below the main `<h1>`. It introduces the contact section. `<h2>` is smaller than `<h1>` but still clearly marks the start of a new section. | Il tag `<h2>` crea un **titolo di secondo livello** — un sottotitolo sotto l'`<h1>` principale. Introduce la sezione dei contatti. `<h2>` è più piccolo di `<h1>` ma indica chiaramente l'inizio di una nuova sezione. |
| `<p>Email: <a href="mailto:sofia.falsiroli@edu.uni-graz.at">sofia.falsiroli@edu.uni-graz.at</a></p>` | A paragraph containing an email link. The `<a>` tag creates a hyperlink. The special `href="mailto:…"` value tells the browser to open the user's email application with the address pre-filled when clicked, making it easy to send a message directly from the page. | Un paragrafo contenente un link email. Il tag `<a>` crea un collegamento ipertestuale. Il valore speciale `href="mailto:…"` indica al browser di aprire il programma email dell'utente con l'indirizzo già compilato al clic, facilitando l'invio di un messaggio direttamente dalla pagina. |
| `<h2>Responsible for Content</h2>` | A second `<h2>` sub-heading introducing the section that names the person legally responsible for the website's content, as required by Austrian and German law. | Un secondo sottotitolo `<h2>` che introduce la sezione in cui è indicata la persona legalmente responsabile del contenuto del sito, come richiesto dalla legge austriaca e tedesca. |
| `<p>Sofia Falsiroli<br>Via Maria Grazia Deledda 4<br>37059 Verona<br>Italy</p>` | A paragraph with the author's name and postal address. The `<br>` tag is a **line break** — it forces the text to continue on a new line without starting a new paragraph. This is the standard way to format addresses in HTML. | Un paragrafo con il nome e l'indirizzo postale dell'autrice. Il tag `<br>` è un **a capo** — forza il testo a continuare su una nuova riga senza aprire un nuovo paragrafo. È il modo standard per formattare gli indirizzi in HTML. |
| `<h2>Disclaimer</h2>` | A third `<h2>` sub-heading introducing the disclaimer section, which clarifies the site's limitations of liability regarding images and third-party content. | Un terzo sottotitolo `<h2>` che introduce la sezione di disclaimer, che chiarisce i limiti di responsabilità del sito riguardo a immagini e contenuti di terze parti. |
| `<p>This site is for informational purposes only. All images and texts are credited …</p>` | The disclaimer paragraph. It states that the site is informational only, that all media are properly credited, and that any copyright concerns can be addressed by contacting the author. | Il paragrafo del disclaimer. Afferma che il sito è solo informativo, che tutti i media sono debitamente accreditati e che eventuali problemi di copyright possono essere risolti contattando l'autrice. |
| `<script src="../js/lang-switch.js"></script>` | A `<script>` tag loads an external **JavaScript** file. JavaScript adds interactive behaviour to the page. The file `lang-switch.js` (located in the `js/` folder one level up) handles language-switching functionality. Placing `<script>` at the **end of the body** — rather than in `<head>` — is best practice: the browser first renders all visible content, and only then loads and runs the script, making the page feel faster. | Un tag `<script>` carica un file **JavaScript** esterno. JavaScript aggiunge comportamento interattivo alla pagina. Il file `lang-switch.js` (nella cartella `js/` un livello sopra) gestisce la funzionalità di cambio lingua. Posizionare `<script>` alla **fine del body** — anziché nell'`<head>` — è una buona pratica: il browser prima visualizza tutto il contenuto, poi carica ed esegue lo script, rendendo la pagina più veloce. |
