# Tutorial: style.css

## Introduction
This tutorial explains the structure and purpose of the `style.css` file, which is the **main stylesheet** of the *Dante's Journey* website. A CSS (Cascading Style Sheets) file controls the visual appearance of every HTML page that links to it — colours, fonts, spacing, layout, and responsive behaviour. Rather than writing individual styles inside each HTML file, all shared visual rules are collected here so that every page on the site looks consistent. The file is divided into several logical sections: a placeholder block for city maps, general body settings, the fixed top banner, a language bar, navigation menus and dropdowns, language-switching controls, content layout, typography, images, and responsive (mobile-friendly) overrides.
