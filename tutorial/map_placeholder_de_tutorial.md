# Tutorial: map_placeholder_de.html

## Introduction
`map_placeholder_de.html` is the **German-language Map page** of the *Dante's Journey* website. Its current purpose is to reserve a place in the navigation for a future interactive map. For now, instead of a real map, the page displays a large pink placeholder box with the label "KARTE PLATZHALTER" (MAP PLACEHOLDER) to show visitors where the map will eventually appear. Because the page has very little unique content, its internal `<style>` block is small — it contains only the rules for the placeholder box. All other layout (navigation, banner, language bar) comes from the shared `css/style.css`.
