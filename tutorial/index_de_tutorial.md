# Tutorial: index_de.html

## Introduction
`index_de.html` is the **German homepage** of the *Dante's Journey* website. It is the first page a visitor sees after choosing German on the language-selection screen (`index.html`). Its job is to welcome the reader, explain the purpose of the project, and give them immediate access to all other sections through the navigation bar. Like every language-specific homepage, it combines the **shared site-wide stylesheet** (`css/style.css`, linked externally) with a small **internal `<style>` block** for rules unique to the homepage text area. A `<script>` tag at the bottom of `<body>` loads the language-switching logic.

| Code | Explanation (English) | Spiegazione (Italiano) |
|------|-----------------------|------------------------|
| `<!DOCTYPE html>` | A **declaration** on the very first line that tells the browser to use the modern HTML5 standard. It is not a visible tag. | Una **dichiarazione** sulla primissima riga che indica al browser di usare lo standard moderno HTML5. Non è un tag visibile. |
| `<html lang="de"> … </html>` | The **root element** that wraps the entire document. `lang="de"` declares German as the page's language, which helps screen readers pronounce text correctly and signals the language to search engines. | L'**elemento radice** che racchiude l'intero documento. `lang="de"` dichiara il tedesco come lingua della pagina, aiutando gli screen reader e i motori di ricerca. |
| `<head> … </head>` | The **invisible** section containing metadata and resources the browser needs before rendering: character encoding, page title, viewport settings, favicon, and stylesheets. | La sezione **invisibile** contenente i metadati e le risorse necessarie prima del rendering: codifica dei caratteri, titolo, impostazioni del viewport, favicon e fogli di stile. |
| `<meta charset="UTF-8">` | Sets **character encoding** to UTF-8, supporting all characters including German umlauts (ä, ö, ü) and Italian accents. | Imposta la **codifica dei caratteri** su UTF-8, supportando tutti i caratteri compresi gli umlauts tedeschi (ä, ö, ü) e gli accenti italiani. |
| `<title>Sulle tracce di Dante</title>` | Sets the **text shown in the browser tab**. The title is kept in Italian ("On the Traces of Dante") because it is the project's proper name, consistent across all language versions. | Imposta il **testo nella scheda del browser**. Il titolo è mantenuto in italiano ("Sulle tracce di Dante") perché è il nome del progetto, coerente in tutte le versioni linguistiche. |
| `<meta name="viewport" content="width=device-width, initial-scale=1.0">` | Makes the page **responsive** on mobile devices: `width=device-width` adapts the page width to the screen, `initial-scale=1.0` sets the default zoom to 100%. | Rende la pagina **responsiva** sui dispositivi mobili: `width=device-width` adatta la larghezza allo schermo, `initial-scale=1.0` imposta lo zoom al 100%. |
| `<link rel="icon" type="image/svg+xml" href="../data/pictures/favicon.svg">` | Loads the **favicon**. The path `../data/pictures/favicon.svg` uses `../` to go one level up from the `de/` folder to reach the shared `data/` directory. | Carica il **favicon**. Il percorso `../data/pictures/favicon.svg` usa `../` per risalire di un livello dalla cartella `de/` e raggiungere la directory condivisa `data/`. |
| `<link rel="stylesheet" href="../css/style.css">` | Links the **shared external stylesheet** that provides site-wide styles (navigation bar, colours, fonts, etc.). `../css/style.css` goes up one level from `de/` then into `css/`. | Collega il **foglio di stile esterno condiviso** che fornisce gli stili del sito (barra di navigazione, colori, caratteri, ecc.). `../css/style.css` risale di un livello da `de/` poi entra in `css/`. |
| `<style> … </style>` | An **internal CSS block** for styles used only on this page — specifically the `.homepage-text` class. | Un **blocco CSS interno** per gli stili usati solo su questa pagina — in particolare la classe `.homepage-text`. |
| `.homepage-text { color: #000; text-align: justify; }` | Styles the main text area. `color: #000` sets the body text to black (overriding any other inherited colour). `text-align: justify` aligns the text to both left and right edges, giving it a neat newspaper-column look. | Stilizza l'area di testo principale. `color: #000` imposta il testo in nero. `text-align: justify` allinea il testo ai bordi sinistro e destro, dando un aspetto ordinato simile a una colonna di giornale. |
| `.homepage-text h2 { color: #b22222; }` | Makes `<h2>` headings inside `.homepage-text` deep red (`#b22222`), the brand colour used throughout the site to maintain visual consistency. | Rende i titoli `<h2>` all'interno di `.homepage-text` di colore rosso scuro (`#b22222`), il colore del marchio usato in tutto il sito per mantenere la coerenza visiva. |
| `<body> … </body>` | Contains all **visible content** of the page. | Contiene tutti i **contenuti visibili** della pagina. |
| `<div class="lang-bar"> … </div>` | A container (`<div>`) for the **language bar** — the thin coloured strip at the top of every page. The `aria-label="Sprachauswahl"` attribute is an **ARIA** (Accessible Rich Internet Applications) label that describes the element to screen readers in German ("language selection"). | Un contenitore per il widget di selezione della lingua. L'attributo `aria-label="Sprachauswahl"` è un'etichetta **ARIA** che descrive l'elemento agli screen reader in tedesco ("selezione della lingua"). |
| `<button id="lang-btn" aria-haspopup="true" aria-expanded="false" aria-label="Sprache wechseln">DE</button>` | The **language button** showing the current language ("DE"). `id="lang-btn"` gives it a unique identifier so JavaScript can control it. `aria-haspopup="true"` tells screen readers that clicking will open a menu. `aria-expanded="false"` indicates the menu is currently closed. `aria-label="Sprache wechseln"` provides the accessible label ("change language"). | Il **pulsante della lingua** che mostra la lingua corrente ("DE"). `id="lang-btn"` gli assegna un identificatore unico per il controllo JavaScript. `aria-haspopup="true"` indica agli screen reader che il clic apre un menu. `aria-expanded="false"` segnala che il menu è chiuso. `aria-label="Sprache wechseln"` fornisce l'etichetta accessibile ("cambia lingua"). |
| `<div id="lang-menu" class="lang-menu" role="menu" aria-label="Verfügbare Sprachen">` | The **dropdown menu** that appears when the language button is clicked. `role="menu"` is an ARIA role identifying this as a menu widget to assistive technologies. `aria-label="Verfügbare Sprachen"` labels it "available languages". | Il **menu a tendina** che appare cliccando il pulsante della lingua. `role="menu"` è un ruolo ARIA che identifica questo come widget menu per le tecnologie assistive. `aria-label="Verfügbare Sprachen"` lo etichetta "lingue disponibili". |
| `<button role="menuitem" data-lang="it">ITA</button>` | A menu item for Italian. `role="menuitem"` identifies it as an item inside the menu (ARIA). `data-lang="it"` is a **data attribute** — a custom attribute that stores the language code. The JavaScript file (`lang-switch.js`) reads this value to navigate to the Italian version of the current page. | Una voce di menu per l'italiano. `role="menuitem"` lo identifica come elemento del menu (ARIA). `data-lang="it"` è un **attributo dati** — un attributo personalizzato che memorizza il codice lingua. | `<div class="main-head"> … </div>` | The **site banner** — the large red bar at the top of every page that shows the site title "Auf Dantes Spuren" (On Dante's Traces). | `<h1>Auf Dantes Spuren</h1>` | The `<h1>` tag creates the **main heading** of the banner. Questo è il titolo più importante della pagina — ogni pagina del sito usa lo stesso `<h1>` come identificatore del marchio coerente. |
| `<header> … </header>` | The semantic `<header>` element wraps the **navigation bar**. Using `<header>` instead of a plain `<div>` is meaningful: it tells the browser and screen readers that this is a site header, improving accessibility and SEO. | L'elemento semantico `<header>` avvolge la **barra di navigazione**. Usare `<header>` invece di un semplice `<div>` è semanticamente significativo: indica al browser e agli screen reader che si tratta di un'intestazione del sito, migliorando accessibilità e SEO. |
| `<nav class="navbar" aria-label="Hauptmenü">` | A semantic `<nav>` element that wraps the navigation links. `aria-label="Hauptmenü"` ("main menu") distinguishes this navigation landmark from others on the page, helping screen reader users jump directly to the navigation. | Un elemento semantico `<nav>` che avvolge i link di navigazione. `aria-label="Hauptmenü"` ("menu principale") distingue questo punto di navigazione dagli altri nella pagina, aiutando gli utenti di screen reader a raggiungerlo direttamente. |
| `<ul class="menu four-cols"> … </ul>` | An **unordered list** (`<ul>`) used to structure the navigation items. `class="menu four-cols"` applies CSS from `style.css` that arranges the four menu items in a horizontal row. | Una **lista non ordinata** (`<ul>`) usata per strutturare le voci di navigazione. `class="menu four-cols"` applica il CSS da `style.css` che dispone le quattro voci in una riga orizzontale. |
| `<li><a href="../index.html">HOME</a></li>` | A list item (`<li>`) containing a navigation link. `href="../index.html"` goes up one folder level to reach the root `index.html` (the language-selection page). | Una voce di lista (`<li>`) contenente un link di navigazione. `href="../index.html"` risale di un livello per raggiungere il file radice `index.html` (la pagina di selezione lingua). |
| `<li class="dropdown"> … </li>` | A list item with `class="dropdown"` that creates the **"Dantes Orte" dropdown menu** (Dante's Places).
