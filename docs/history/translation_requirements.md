1. Natural language 
    ○ The translation must sound natural in English, not literal. 
    ○ Paragraphs should be actively reformulated without changing the information. 
2. Preserve Markdown 
    ○ Keep all headings, subheadings, paragraphs, bold, italics, lists, and links exactly as in the original Markdown. 
        ○ Do not add extra horizontal lines between paragraphs. 
3. Italian terms 
    ○ Names of places, institutions, religious orders, etc., remain in Italian and should be in italics, except for the main city name (e.g., Bologna), which remains normal. 
4. Quotations from the Divina Commedia 
    ○ Do not translate the quotations. 
    
    ○ Sources for the quotes should have the heading:
        #### Sources for quotes from the *Divina Commedia* 
    
5. Source links 
    ○ Include direct URLs, not site names, for both text sources and quotation sources. 
 
6. Output ready for a bilingual website 
        ○ Markdown structure must be ready to use directly on the site. 
Easy to update with new texts or translations without modifying HTML.
