## KONZEPT FÜR ABSCHLUSSPROJEKT

**TITEL:** Die Orte von Dante

**BESCHREIBUNG:**  
Mein Projekt ist eine Webseite, die mittels einer interaktiven Karte zeigt, die Orte (Städte aber auch Regionen), wo Dante (1265-1321) während seinem Exil gelebt hat.  
Meine Idee:  
- Main Page: interaktive Karte   
- Wenn man auf die Ikone der Stadt klickt, öffnet sich eine neue Seite die:  
  - genauere Informationen zu dem Aufenthalt Dantes zeigt (Zeit, Gastgeber, eventuelle Verbindungen mit der *Commedia*)  
  - Informationen über einen Besuch der Stadt gibt (gibt es Statuen/Museen/Sehenswürdigkeiten, die Dante gewidmet sind, Öffnungszeiten, Tickets usw...)  
- Dreisprachige Webseite: Italienisch (original), Englisch (Übersetzung durch KI), Deutsch (Übersetzung durch KI)

**FÜR WEN:**   
Für jede Person die Interesse hat, Dantes Spuren in Italien zu folgen, sowohl digital als auch vor Ort 

**INHALTE:**  
- Texte (von mir oder KI Übersetzung)  
- API für die interaktive Karte  
- Bilder (vermutlich) von Wikimedia Commons
